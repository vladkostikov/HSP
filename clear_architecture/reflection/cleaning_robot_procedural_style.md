# Рефлексия по ../cleaning_robot_procedural_style.py

Задание состояло в написании программы для дистанционного управления роботом.
Реализацию нужно было выполнить с помощью функций, без использования классов.

## Преимущества решения

- удобная реализация create_robot
- только одна публичная функция call_robot
- код стал короче по сравнению с предыдущим решением
- простая логика

## Недостатки решения

- не вынесена в отдельную функцию валидация
- не вынесено в отдельную функцию взаимодействие с роботом
